Capstone Story

Problem

Predict housing prices using a large real estate dataset to support better decision-making.

Data Journey

Dataset contained over 2 million rows with missing values and outliers. Required cleaning, transformation, and encoding.

Solution

Used Python, performed EDA, applied median imputation, and built a Random Forest model with an R² of approximately 0.30.

Impact

Identified how data quality affects predictions and proposed improvements such as feature engineering and better imputation.
