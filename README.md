🧠 Capstone Project Summary
Problem

Develop a model to predict housing prices using a large real estate dataset.

Data Journey
Dataset contains over 2 million records
Missing values (~25%) in key variables
Presence of outliers in price and house size
Required extensive data cleaning and preprocessing
Solution
Performed Exploratory Data Analysis (EDA)
Applied median imputation for missing values
Encoded categorical variables
Built a Random Forest regression model
Evaluated performance using R² (current ≈ 0.30)
Impact
Identified how data quality affects model performance
Demonstrated importance of EDA in machine learning workflows
Proposed improvements such as feature engineering and advanced imputation methods
